# AVR
Модули для работы с микроконтроллерами AVR. В этой ветке все настроено для работы в программе Microchip Studio.

Модули можно подключать и отключать в файле Config.inc в главной ветке программы. Далее читаем файлы README для каждого, нужного вам модуля и настраиваем его.

Так же приветствуются любые улучшения кода, которые на ваш взгляд помогут работать модулям более стабильно и быстрее.

Всем спасибо.
